###########################
xarray-bounds documentation
###########################

**Version:** |version|

A :py:class:`xarray.Dataset` accessor for managing `CF <https://cfconventions.org/Data/cf-conventions/cf-conventions-1.12/cf-conventions.html#cell-boundaries>`_-compliant boundary variables.

..  warning::

    This project is under active development. Frequent and breaking changes are expected.

    Significant breaking changes are anticipated with :py:mod:`xarray`'s upcomming support for CF-compliant interval indexes.
    See https://github.com/pydata/xarray/pull/10296 for more information.
